“แบบสอบถามความพึงพอใจการใช้งานของ สปสช.”

  1. แบบสอบถามความพึงพอใจการใช้งานของ สปสช. 
  2. ระบบรวบรวมข้อมูล ปัญหาการเข้าใช้งานโปรแกรมประยุกต์ของ สปสช.
  3. (ร่าง) ระเบียบคณะกรรมการหลักประกันสุขภาพแห่งชาติ ว่าด้วย ประมวลจริยธรรมสำหรับคณะกรรมการ ผู้บริหาร และผู้ปฏิบัติงาน ตามกฎหมายว่าด้วยหลักประกันสุขภาพแห่งชาติ พ.ศ. ๒๕๕๖

การบริหารจัดการรายโรค

  1. ลงทะเบียนผู้ป่วยรายโรค (DMIS) 
  2. โครงการบูรณาการระบบสารสนเทศการให้บริการผู้ติดเชื้อ/ผู้ป่วยเอดส์(NAP) 
  3. ระบบสารสนเทศเพื่อการให้บริการผู้ป่วยไตวายเรื้อรังระยะสุดท้าย 
  4. รายงานสถิติการลงทะเบียนผู้ป่วยรายโรค (DMIS) 
  5. ระบบข้อมูลบริการผู้ป่วยโรคหืดและโรคปอดอุดกั้นเรื้อรัง (Smart AsCOPD) 
  6. ระบบสารสนเทศให้บริการผู้ป่วยวัณโรค (TB DATA HUB)

งานด้านอื่นๆ

  1. ระบบหลักประกันสุขภาพในระดับท้องถิ่นหรือพื้นที่ (อบต.)  
  2. ระบบรายงานเงินกองทุนและกิจกรรมสำนักงานสาขาจังหวัด 
  3. อนุกรรมการตรวจสอบรับข้อร้องเรียน 
  4. เว็บไซต์หลักระบบองค์การการปกครองส่วนท้องถิ่น (อปท.) 
  5. ระบบประมวลผลข้อมูลผู้ป่วยนอกสิทธิประกันสังคม (SS-DATA) 
  6. ระบบบริหารจัดการกองทุนฟื้นฟูสมรรถภาพที่จำเป็นต่อสุขภาพระดับจังหวัด (กองทุน อบจ.) 

 แผนงานสนับสนุนระบบบริการฟื้นฟูสมรรถภาพด้านการแพทย์

  1. การบริหารจัดการค่าใช้จ่ายบริการฟื้นฟูสมรรถภาพด้านการแพทย์ 
  2. ระบบรายงานอุปกรณ์คนพิการและการให้บริการฟื้นฟูสมรรถภาพ : โปรแกรมบันทึกข้อมูลเบิกอุปกรณ์คนพิการและการให้บริการฟื้นฟูสมรรถภาพ 
  3. โครงการพัฒนาระบบบริการอุปกรณ์เครื่องช่วยฟังไทยสำหรับคนพิการทางการได้ยิน ภายใต้ระบบหลักประกันสุขภาพแห่งชาติ 

งานส่งเสริมสุขภาพและป้องกันโรค

  1. ระบบกองทุนสร้างเสริมสุขภาพและป้องกันโรค 
  2. ระบบบริหารจัดการโรคธาลัสซีเมีย 
  3. ระบบติดตามผลการให้ยาไทรอยด์สำหรับเด็กแรกเกิด 

งานตรวจสอบการชดเชยและคุณภาพบริการ

  1. ระบบบันทึกและตรวจประเมินคุณภาพการบันทึกเวชระเบียน (MRA) 
  2. ระบบอุทธรณ์ออนไลน์ 
  3. ระบบตรวจสอบเวชระเบียนอิเล็กทรอนิกส์ (eMA) 
  4. ระบบบันทึกข้อมูลด้านคุณภาพการรักษาของโรคมะเร็งเต้านม มะเร็งลำไส้และมะเร็งปอด (Auditcancer)

  3. รายงานการจ่ายเงินกองทุน
